Welcome to kube-boot Helm Charts

Thank you for visiting the official Helm charts repository for kube-boot. Below you'll find instructions on how to add this Helm repository to your local Helm setup and use the available charts.

Prerequisites

  • Helm (version 3.x or later)
  • Access to this GitHub repository or the hosted version (via GitHub Pages or another Helm repository host).

Adding the Helm Repository

To add the Helm repository, run the following command:

        helm repo add kube-boot https://borntoprogram.github.io/helm-charts/

This will add the kube-boot Helm repository to your local Helm setup. You can then search for available charts by running:

        helm search repo kube-boot

Installing Charts

To install the kube-boot Helm chart, use the following command:

        helm install my-kube-boot-release kube-boot/kube-boot --version <version>

Replace <version> with the desired version, for example, 1.0.0.

Updating the Helm Repository

If the Helm repository has been updated, make sure to refresh your local copy with the following command:

        helm repo update

Known Issues

  • Error adding the repository: Ensure you are using the correct URL and that the Helm repository has been properly published (e.g., via GitHub Pages).
  • Invalid index.yaml: Check that the index.yaml file is present in the repository root and correctly references chart paths.

Development and Contributions

If you'd like to contribute to this repository or add your own Helm charts, fork this repository and send a pull request.

If you have any questions or issues, please reach out to the repository team via GitHub Issues.
